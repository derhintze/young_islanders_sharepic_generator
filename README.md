# EV Lindau Young Islanders Sharepic Generator

This tool creates sharepics for EV Lindau Young Islanders youth ice hockey teams from
Lindau, Germany.

To run the tool, you'll need the following static assets:

  * the EV Lindau Islanders background (stored as `bckgrnd.jpeg` in the source root)
  * the EV Lindau Islanders logo (stored as `evl-logo_ohne-kontur.svg` in the source
    root)
  
Furthermore, you'll need the font Industry installed on your system.
